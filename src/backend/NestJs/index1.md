# nest 学习

## 安装 nest  cli

```bash
# 安装 nest cli 工具
npm install -g @nestjs/cli
yarn global add @nestjs/cli
pnpm install -g @nestjs/cli
```

## 创建项目

```bash
nest new project-name # 创建项目

```

## nest cli 常用命令

```bash
nest --help  # 查看帮助信息

nest generate module module-name   # 生成模块
nest generate controller controller-name # 生成控制器
nest generate service service-name # 生成服务
```
